# NATHANFILLIONLE
A wordle where the answer is NATHANFILLION 90% of the time and 10% of the time it could be a 13-letter word

## Inspiration
This project is inspired by jeffgoldblumle.com, where 90% of the time the answer is JEFFGOLDBLUM and 10% of the time it is another 12-letter word

## How to play
Go to [insert URL] and start typing to guess. Press enter to guess the word you input.
